# Amazon IoT TwinMaker (amazon-iot-twinmaker)
AWS IoT TwinMaker makes it easier for developers to create digital twins of real-world systems such as buildings, factories, industrial equipment, and production lines. It enables you to optimize operations by easily creating virtual representations that help visualize and monitor physical environments.

## APIs

### AWS IoT TwinMaker API
The AWS IoT TwinMaker API provides programmatic access to create and manage workspaces, entities, components, scenes, and sync jobs for building digital twin representations of physical environments.
